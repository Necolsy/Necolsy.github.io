# Subin Lin

Personal academic website for Subin Lin, PhD Candidate in Built Environment at the National University of Singapore.

The site includes a short bio, research interests, news, peer-reviewed publications, an academic timeline, and travel notes. It is built with the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme and published through GitHub Pages.

## Update the site

- Edit `_pages/about.md` for the homepage.
- Add or update publications in `_bibliography/papers.bib`.
- Add short updates in `_news/`.
- Update links in `_data/socials.yml`.
